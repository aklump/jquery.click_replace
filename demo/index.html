Demo of jquery.click_replace.js

  • For no js fallback, set display: none on all click elements in a stylesheet.
  • Notice the size of #div1 and #div3 are taken from their replacement targets.
  • Notice the dynamic classes added to the elements.
  • It is possible to go back in the first example; view source for how to.
  • The second example shows how to pass the replace element as a jquery selector via the data-replace attribute.
